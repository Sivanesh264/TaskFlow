# TaskFlow Frontend

A modern and responsive Task Management Application frontend built with React.js and Vite.

## Features

- User Registration & Login
- JWT Authentication
- Protected Routes
- Dashboard Analytics
- Task Management Interface
- Create, Edit, Delete Tasks
- Task Search & Filtering
- User Profile Management
- Dark / Light Theme
- Responsive Design
- Toast Notifications

## Tech Stack

- React.js
- Vite
- React Router DOM
- Axios
- Tailwind CSS
- React Hot Toast
- Context API

## Installation

1. Navigate to frontend directory

```bash
cd frontend
```

2. Install dependencies

```bash
npm install
```

3. Start development server

```bash
npm run dev
```

4. Open browser

```text
http://localhost:5173
```

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=http://localhost:5000/api
```

## Available Scripts

```bash
npm run dev
```

Runs development server.

```bash
npm run build
```
